# 全球超市可视分析与自动设计项目

### 前言

项目分为可视分析大屏展示与可视化图表自动设计

项目引用了 `observablehq` 项目内模块：

- [Map @Freell](https://observablehq.com/@1906030114/map)

- [Hello Draco @leyoubaloy](https://observablehq.com/d/c46ba76a55f54991)

### 运行

在目录使用 `npm start`

#### Windows用户

运行 `run.bat`

#### Linux用户

运行 `run.sh`
